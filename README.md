# ⏱️ Luxury Circular Stopwatch

A modern, visually appealing stopwatch built using **HTML, CSS, and JavaScript**, featuring a **circular watch-style UI** with premium color themes like gold, dark blue, and black.

---

## ✨ Features

* 🔵 **Circular Stopwatch Design** – Inspired by classic luxury watches
* 🎨 **Premium Color Theme** – Gold, dark blue, black gradients
* ✨ **Animated Glow Effects** – Pulsing and rotating border animation
* ▶️ **Start / Stop / Reset / Lap Functions**
* 📋 **Lap Time Tracking** with styled list
* 💎 **Glassmorphism UI** for a modern look
* 📱 **Responsive Centered Layout**

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & Animations
* **JavaScript** – Stopwatch Logic

---

## 🎯 UI Highlights

* Circular dial with **animated gold ring**
* Smooth **hover effects on buttons**
* Elegant **lap display with subtle color variations**
* Fully centered layout using **Flexbox**

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start using the stopwatch

---

## 🎮 Controls

* ▶️ **Start** – Begins the timer
* ⏹️ **Stop** – Pauses the timer
* 🔄 **Reset** – Resets time to zero
* 🏁 **Lap** – Records current time
